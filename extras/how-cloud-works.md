# How Cloud Works

## 1. What is Cloud?
- Cloud = renting computing over internet.
- No hardware ownership.
- Pay-as-you-go.

## 2. CAPEX vs OPEX
### CAPEX (Capital Expense)
- Buy servers upfront.
- High cost.
- Hard to scale.

### OPEX (Operational Expense)
- Rent resources.
- Pay per use.
- Easy scaling.

## 3. Virtualization
- Hypervisor divides physical server into multiple VMs.
- Foundation of AWS EC2.

## 4. Cloud Models
- IaaS: EC2, VPC, S3
- PaaS: RDS, Beanstalk
- SaaS: Gmail, Salesforce

## 5. Why companies use cloud
- Cheaper
- Faster
- Scalable
- No hardware maintenance
