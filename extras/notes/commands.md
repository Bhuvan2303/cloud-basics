# Linux Filesystem Basics

## Root Directory "/"
Everything starts from here.

## Important Folders
- /home → user files
- /root → admin user files
- /etc → system config files
- /bin → basic commands
- /sbin → admin commands
- /var → logs & app data
- /usr → user applications
- /tmp → temporary files

## Paths
- Absolute path: starts with "/"
- Relative path: based on current folder

## Common Commands
ls, cd, mkdir, touch, cp, mv, rm, cat, pwd
