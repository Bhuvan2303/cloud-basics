## Terraform
Terraform is an Infrastructure as Code tool
used to manage cloud resources declaratively.

---

## Provider
A provider connects Terraform to a cloud platform
and enables resource management.

---

## Resources
Resources represent actual infrastructure
such as virtual machines, networks, and load balancers.

---

## Plan and Apply
The plan command previews changes.
The apply command executes infrastructure updates.

---

## State
Terraform maintains a state file to track
existing resources and manage updates safely.
