## Logs
Logs record system and application events.
They are essential for debugging, security, and incident analysis.

---

## System Logs
System logs store operating system events such as errors,
service failures, and startup messages.

---

## Authentication Logs
Authentication logs track login attempts and sudo activity.
They are critical for security monitoring.

---

## journalctl
journalctl is used to view logs managed by systemd.
It allows filtering logs by service and time.

---

## Operational Importance
Logs help engineers identify root causes instead of guessing.
Effective monitoring depends on reliable logging.
