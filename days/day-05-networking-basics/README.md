# Day 5 – Networking Basics

## Network
A network is a group of computers connected to share data and resources.

## IP Address
An IP address is a unique numerical identifier assigned to a device on a network.
It allows devices to identify and communicate with each other.

Example:
192.168.1.10

## Public IP vs Private IP
Private IP:
- Used inside a private network
- Not accessible from the internet

Public IP:
- Assigned by ISP or cloud provider
- Accessible over the internet

## Port
A port identifies a specific service or application running on a computer.
It allows multiple services to use the same IP address.

Common ports:
- 80  : HTTP
- 443 : HTTPS
- 22  : SSH

## DNS
DNS (Domain Name System) converts human-readable domain names into IP addresses.
Example:
google.com → IP address
