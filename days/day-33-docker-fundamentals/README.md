## Docker
Docker is a container platform used to package and run applications
with their dependencies in a consistent environment.

---

## Images and Containers
Docker images are read-only templates.
Containers are running instances of images.

---

## Dockerfile
A Dockerfile defines the steps to build a Docker image.
It includes base image, dependencies, and startup command.

---

## Core Commands
Docker provides commands to build images,
run containers, and manage their lifecycle.

---

## Benefits
Docker enables fast deployments,
consistent environments, and efficient resource usage.
