## Secrets
Secrets are sensitive values such as passwords,
API keys, and tokens that grant access to systems.

---

## Risks of Hardcoding
Hardcoding secrets in code or repositories
can lead to leaks and security breaches.

---

## Secret Management Approaches
Secrets can be managed using environment variables,
protected configuration files, or dedicated secret managers.

---

## Secret Managers
Secret managers store secrets securely,
control access using IAM, and support rotation.

---

## Best Practices
- Never commit secrets to version control
- Restrict access to secrets
- Rotate secrets regularly
